## 概念

當我們希望程式根據「不同情況執行不同結果」時，就需要使用條件判斷！

## 基本語法結構

{% raw %}

```python
if 條件1:
    # 條件1為真，執行這區塊
elif 條件2:
    # 條件1為假且條件2為真
else:
    # 上述條件都不成立時

```

## 邏輯運算符（and / or / not）

| 符號  | 意義 | 範例                  |
| ----- | ---- | --------------------- |
| `and` | 且   | `x > 3 and x < 10` ✅ |
| `or`  | 或   | `x < 0 or x > 10` ❌  |
| `not` | 取反 | `not x == 5` ❌       |
