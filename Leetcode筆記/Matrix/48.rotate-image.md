## 說明

他是要原地修改，不可以使用額外的空間

## 解法

-   STEP01：先上下翻
-   STEP02：主對角線轉置（只轉上三角）

```cpp
class Solution {
public:
    void rotate(vector<vector<int>>& matrix) {
        int m = matrix.size();
        int n = matrix[0].size();
        // STEP01
        for(int i=0;i<m/2;i++){
            for(int j=0;j<n;j++){
                swap(matrix[i][j], matrix[m-i-1][j]);
            }
        }
        // STEP02
        for(int i=0;i<m;i++){
            for(int j=i;j<n;j++){
                swap(matrix[i][j], matrix[j][i]);
            }
        }
    }
};
```
